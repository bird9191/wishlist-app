Wishlist App

Веб-приложение для создания списков желаний. Пользователь создаёт вишлист, добавляет товары и делится ссылкой с друзьями. Друзья могут зарезервировать подарок или скинуться на дорогой — владелец не видит кто и сколько, чтобы сюрприз не испортился.

Стек: Next.js, React, TypeScript, Tailwind на фронте. FastAPI, SQLAlchemy, PostgreSQL на бэке. WebSocket для real-time обновлений. JWT авторизация. Мобильный клиент на Flutter/Dart.

Как запустить

Бэкенд:
  cd backend
  pip install -r requirements.txt
  uvicorn app.main:app --reload --port 8000

Фронтенд:
  cd frontend
  npm install
  npm run dev

Мобилка:
  cd mobile_flutter
  flutter pub get
  flutter run

Что умеет

Секретное резервирование. Друг нажимает "зарезервировать", товар помечается как занятый, но владелец не знает кем.

Коллективные сборы. Если подарок дорогой, можно включить режим сбора. Несколько человек вносят вклады, прогресс-бар показывает сколько собрано. Владелец видит только общую сумму, но не кто сколько дал.

Real-time. Все изменения приходят мгновенно через WebSocket, без перезагрузки страницы.

Автозаполнение по URL. Вставляешь ссылку на товар, приложение само подтягивает название, картинку и цену.

Публичные ссылки. Друзья открывают вишлист без регистрации. Аккаунт нужен только для создания списка.

Продуктовые решения

Товар с активными вкладами нельзя просто удалить. Минимальный вклад 1 рубль, нельзя внести больше оставшейся суммы. Резервацию можно отменить по email. Пустой вишлист показывает подсказки.

Владелец и гость видят разные данные — это сделано через отдельные Pydantic-схемы на бэкенде (WishlistOwner и WishlistGuest).

API URL для мобильного клиента задаётся в mobile_flutter/lib/config.dart.
